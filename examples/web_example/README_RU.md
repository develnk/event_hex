# Event Hex Examples

## Описание

Демонстрация проектов построенных на гексагональной архитектуре, использующие **Event Sourcing**, **CQRS**, **DDD**, *
*TDD** с
использованием **event_hex** библиотеки в качестве ядра архитектуры.

## Технологический стек

- **Язык**: Rust 1.85+ (Edition 2024)
- **HTTP API**: `salvo` 0.89.3
- **База данных**: MongoDB 3.5.2
- **Async**: `tokio` 1.50 + `async-trait`
- **Сериализация**: `serde`, `erased-serde`, `bincode`
- **Логирование**: `tracing` 0.1.44
